# 天龙八部多开启动器 (TLBB Launcher)

一个基于 Python 编写的现代化、支持无限多开的《天龙八部》私服/官服通用辅助启动器。

## ✨ 核心特性

- **🚀 无限多开机制**：底层彻底重构了网络代理端口的防占逻辑。不论你想开两个号还是十个号，只需反复点击启动即可，各个客户端的网络通道绝对隔离，**互不影响**。
- **🛡️ 后台隐身代理守护**：彻底摆脱传统启动器“主黑框绝对不能关”的痛点！所有的网络端口转发逻辑全部在系统级后台静默运行。你每次点完启动，界面就会直接销毁**即用即走**。哪怕你强行关闭或杀死某个游戏窗口，也绝对不会牵连其他在线的客户端断线。
- **🖥️ 可视化图形界面 (GUI)**：告别繁琐且容易出错的代码修改！提供清晰直观的配置界面，轻松配置游戏安装路径、服务器 IP 及其配套的登录和游戏端口。
- **💾 智能配置自动记忆**：你在面板上填写的任何配置参数，在点击启动瞬间会自动保存到同目录下的 `launcher_config.json` 文件中，下次启动直接读取，一步到位。
- **🛑 一键强杀清场**：面板自带实时的“代理雷达监控”和“一键结束所有后台代理”按钮，方便你在挂机结束后进行彻底的网络重置与进程清场。

## 🛠️ 运行环境

本工具秉持了“极简主义”，完全基于 Python 标准库编写，**零第三方依赖**，克隆下来即拷即用：
- Python 3.x
- `tkinter` (Python 内置 GUI 界面库，无需 pip 安装)

## 🎮 使用指南

### 1. 源码直接运行 (开发者推荐)

在含有 Python 环境的电脑上，直接运行该脚本：
```bash
python launcherByPython
```

1. 在弹出的界面中，点击 **浏览** 选取你的游戏根目录。
2. 填写正确的 **服务器 IP** 和对应的 **登录端口/游戏端口**。
3. （可选）如果你有特殊的启动指令，可在 **启动参数** 栏进行调整。
4. 点击 **🚀 启动游戏 (支持多开)**。随后界面会自动隐藏，后台默默接管一切网络转发任务。多开只需重新运行脚本再次点击！

### 2. 打包为免环境可执行文件 (.exe)

如果你想把启动器发给没有安装 Python 的朋友，或者单纯追求纯净的双击体验，可以通过以下步骤将其打包为 `.exe` 文件（**必须在 Windows 环境或虚拟机下操作**）：

1. 打开 CMD 或 PowerShell，安装打包神器 `pyinstaller`：
   ```bash
   pip install pyinstaller
   ```
2. 在该脚本所在目录下，运行终极无痕打包指令：
   ```bash
   pyinstaller -F -w launcherByPython
   ```
   *(注：`-F` 表示打包为纯净的单文件，`-w` 表示剥离所有黑框控制台)*
3. 稍等片刻，在生成的 `dist` 文件夹内就能找到 `launcherByPython.exe` 了。你可以把它重命名为 `天龙多开器.exe`，直接放到任何位置双击畅玩！

## 📄 常见问题与注意事项

- **怎么彻底关闭它？**：代理程序经过特殊设计，极其轻量化。如果强行需要关闭代理中枢，直接在启动器界面点击“一键结束后台代理”即可。
- **报毒说明**：使用 PyInstaller 编译的单文件 EXE 及带有 Socket 隐形网络代理逻辑的程序，极易被 360/Windows Defender 判定为异常行为。请放心在杀软中将其添加至信任白名单，本仓库源码全公开透明，绝无任何恶意逻辑。
